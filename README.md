# Input Control + Log & Toast App

Aplikasi Android sederhana yang menampilkan penggunaan Date Picker dan Alert Dialog.  
Pengguna dapat memilih tanggal dan melihat hasilnya ditampilkan pada layar.
Update terbaru yaitu menambahkan logcat & toast

## Fitur
- Menampilkan tanggal yang dipilih menggunakan Date Picker
- Menampilkan pesan konfirmasi menggunakan Alert Dialog
